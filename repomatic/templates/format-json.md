### Description

Auto-formats JSON files with [Biome](https://biomejs.dev/). See the [`format-json` job documentation](https://kdeldycke.github.io/repomatic/workflows.html#github-workflows-autofix-yaml-jobs) for details.

> [!TIP]
> Customize JSON formatting via [`[tool.biome]`](https://biomejs.dev/reference/configuration/) in your `pyproject.toml`, or via a native `biome.json` file.
