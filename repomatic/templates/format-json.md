### Description

Auto-formats JSON files with [Biome](https://biomejs.dev/). See the [`format-json` job documentation](https://github.com/kdeldycke/repomatic?tab=readme-ov-file#githubworkflowsautofixyaml-jobs) for details.

> [!TIP]
> Customize JSON formatting via [`[tool.biome]`](https://biomejs.dev/reference/configuration/) in your `pyproject.toml`, or via a native `biome.json` file.
